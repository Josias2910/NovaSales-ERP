# 🚀 NovaSales ERP - Sistema de Gestión Comercial
![Estado](https://img.shields.io/badge/Estado-Functional-green) ![Lenguaje](https://img.shields.io/badge/Lenguaje-C%23-blue) ![Framework](https://img.shields.io/badge/.NET-Framework%204.8-purple) ![DB](https://img.shields.io/badge/DB-SQL%20Server-red)

**NovaSales ERP** es una solución robusta de escritorio diseñada para centralizar y optimizar las operaciones de ventas, compras y control de inventario en entornos comerciales medianos y pequeños.

## 🛠️ Stack Tecnológico
* **Lenguaje:** C# con .NET Framework 4.8.
* **Arquitectura:** Diseño en **N-Capas** (Presentación, Negocio, Datos, Entidades) para máxima escalabilidad y mantenimiento.
* **Persistencia:** Entity Framework Core / LINQ para consultas optimizadas.
* **Base de Datos:** Microsoft SQL Server.
* **Seguridad:** Encriptación de credenciales y sistema de autenticación por roles.
* **Despliegue:** Generación de instaladores profesionales mediante **Inno Setup**.

---

## 📌 Características Principales
* **Control de Inventario:** Monitoreo automático de stock con alertas de faltantes.
* **Módulo de Ventas y Compras:** Registro histórico completo con cálculo de impuestos y márgenes de ganancia.
* **Generación de Comprobantes:** Exportación nativa a formatos **PDF** y hojas de cálculo **Excel**.
* **Gestión de Usuarios:** Permisos diferenciados según el perfil del empleado (Admin/Empleado).
* **Estadísticas en Tiempo Real:** Visualización gráfica de los productos más vendidos y flujo de caja.

---

## 🏗️ Detalles Técnicos de Implementación
* **Capa de Datos:** Implementación del patrón *Repository* para desacoplar la lógica de acceso a datos.
* **Capa de Negocio:** Centralización de las reglas de validación (precios, stock mínimo, duplicados) antes de la persistencia.
* **Configuración de Entorno:** El sistema incluye scripts de migración que configuran automáticamente la instancia local de **SQL Express** durante la instalación.
